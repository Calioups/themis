About Themis

About this blog

Themis is a hybrid digital-analog music synthetiser based on analog chips CEM3340, CEM3320 and V2164D by CoolAudio, a Raspberry Pi 3 for the user interface, and a STM32F7 for the enveloppes generation. The project is hosted by ENSEA Graduate School of Electrical Engineering, Cergy, Univ. Paris-Seine, and involves a dozen graduate students each year. A working prototype is to be released by Spring 2019. The project is led by Professor S. Reynal.
